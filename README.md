###  Foco: Dados, Matemática & Performance

Sou um desenvolvedor com base forte em **C/C++**, migrando meu foco para **Engenharia de Dados**.  
Gosto de olhar código como se fosse um teorema: entender a prova, a estrutura e a complexidade.

> $$f(\text{dev}) = \text{dados} + \text{matemática} + \text{performance}$$

-  Interesse em: Engenharia de Dados, ETL, modelagem, qualidade e fluxo de dados
-  Background em: C, C++, C#, Python, algoritmos, estruturas de dados, teoria dos grafos
-  Gosto de temas como: complexidade, lógica, teoria da computação e probabilidade

###  Matemática no código

Algumas ideias que curto levar para o que programo:

- **Estruturas de dados** vistas como objetos matemáticos (árvores, grafos, filas, etc.)
- **Complexidade assintótica** como forma de medir “elegância” de uma solução
- **Probabilidade e estatística** para trabalhar com dados reais
